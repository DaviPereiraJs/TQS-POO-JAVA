# 📑 Resumo de Aprendizado: Conceitos de POO em Java

Este repositório contém exercícios práticos para o domínio dos pilares da Programação Orientada a Objetos. Abaixo está o resumo das competências desenvolvidas em cada questão:

| Questão | Foco Principal | O que foi desenvolvido |
| :--- | :--- | :--- |
| **1. Employee (Básica)** | **Classes e Objetos** 🏗️ | Transformação de conceitos do mundo real em código Java, definindo atributos e métodos básicos. |
| **2. Construtores** | **Inicialização** ✨ | Implementação de métodos construtores para garantir que os objetos sejam criados com valores válidos desde o início. |
| **3. Invoice** | **Encapsulamento** 🛡️ | Uso de modificadores de acesso `private` e métodos `get`/`set` para proteção de dados e validação de regras de negócio. |
| **4. Employee (Avançada)** | **Lógica de Negócio** 📈 | Criação de métodos internos para manipulação de estados, como cálculos de reajuste salarial e processamento de dados. |
| **5. Ingresso/VIP** | **Herança** 🧬 | Aplicação de reutilização de código, estendendo funcionalidades de uma classe "mãe" para uma classe "filha" especializada. |
